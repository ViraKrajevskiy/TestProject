Демонстрация древовидного меню Django App

Главное меню (main_menu):

{% load menu_tags %} {% draw_menu 'main_menu' %}

Информация о текущей странице:

URL: {{ request.path }}

Заголовок: Демонстрационная страница

Переходите по разным пунктам меню чтобы увидеть, как работает выделение активного пункта и автоматическое раскрытие подменю.

Особенности реализации:

  • Всего 1 запрос к БД для отрисовки меню
  • Автоматическое определение активного пункта по URL
  • Разворачивание родителей активного пункта и его детей первого уровня
  • Поддержка named URL и явных URL
  • Поддержка нескольких меню на одной странице
  • Редактирование через стандартную админку Django
